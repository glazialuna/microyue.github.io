cv2

install

  • conda:conda install -c https://conda.binstar.org/menpo opencv

    函数

  1. 色彩转换：
    • img_gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY) # BGR转灰度
    • img_bgr = cv2.cvtColor(img_gray, cv2.COLOR_GRAY2BGR) # 灰度转BRG
    • img_rgb = cv2.cvtColor(img_gray, cv2.COLOR_GRAY2RGB) # 灰度转RGB
    • b,g,r = cv2.split(img) #bgr图像分离三个通道
    • img2 = cv2.merge([r,g,b]) #merge成rgb图像

swig

  • install swig package (win) download
  • unpack zip in a path
  • add path to system variable PATH
  • check : cmd -> swig -version

pypylon

注意pypylon 只匹配 pylon 5.2.0 (截止2019.10.12) , 安装developer才有sdk包
pypylon 是辅助 Basler camera 系列的python API接口

  • pre: pylon , python , pip ,pyplon

  • install (fron wheel):

    • download wheel(builded) for your system, release
    • install wheel :pip install XXX/X/XX.wheel

  • install (from source)

    • prepare version (swag > 3.0.12 , python > 3.4)

    • install :

    • build in python : python setup.py develop

  • test: in file tests/
