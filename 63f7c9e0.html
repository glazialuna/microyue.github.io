安装包时遇到问题

  1. pip._vendor.urllib3.exceptions.ReadTimeoutError: HTTPSConnectionPool(host='files.pythonhosted.org', port=443): Read timed out.

    这个是在网络状态不佳时出现的问题，解决方法是修改默认的timeout(默认100)，如

    pip install spacy
    pip install –default-timeout=1000 spacy

  2. Could not install packages due to an EnvironmentError: [WinError 5] 拒绝访问。: 'c:\\programdata\\anaconda3\\lib\\site-packages\\importlib_metadata-0.6.dist-info\\INSTALLER' Consider using the–useroption or check the permissions.

    解决方法：

    pip install spacy
    pip install spacy –user

  3. OverflowError: Python int too large to convert to C long

    解决方法:修改torchtext的utils.py

    • csv.field_size_limit(sys.maxsize)
    • csv.field_size_limit(int(ctypes.c_ulong(-1).value // 2))
    • import ctypes
