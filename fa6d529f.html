Installation

Step 1： 安装rust语言包和Cargo工具包

  • 安装rust语言编译环境
    • 安装：官网
    • 按照exe的提示进行操作
  • 安装cargo工具包(用于管理crated等工具包)
    • 安装完成rust后，利用rustup install beta安装cargo
    • 非windows用户参考Cargo Book
  • 开始使用cargo的例子：
    • 建立新的项目
      cargo new hello_world
    • 查看内容
      • 项目文件中cargo.toml配置文件
      • 源代码在src中
    • 编译项目
      cargo build
    • 运行可执行文件
      cargo run

Step 2： 配置VS code

  • extension 中搜索rust安装
  • 想用run快捷键暂时还没搞出来，只能terminal里面用cargo run命令

crates使用

  • 在cargo.toml文件中的dependencies下添加
    • ring=1.0.0
    • 版本号或是利用git查询crates.io
  • 使用直接利用use即可
    • use ring::digest

      添加derive

  • 在目标前添加
