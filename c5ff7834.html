SSH 链接

  • 下载git 工具
  • 绑定用户

    git config –global user.name “username”
    git config –global user.email “usermail@163.com“

  • SSH 链接
    • ssh-keygen -t rsa 生成密钥
    • cd ~/.ssh 查看文件密钥生成（3个文件）复制id_rsa.pub
    • 打开github网页 -> setting -> SSH菜单 -> Add SSH key ,粘贴 id_rsa.pub文件

链接github的repo和pycharm项目

到pycharm项目->setting->Version Control -> Github添加github账号，验证通过。

然后在当前项目中->VCS->Import into Version Control -> Share Project on Github

选择合适的文件commit到项目中即可，之后保持同步只需要VCS-commit同步更新即可
