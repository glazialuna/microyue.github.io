课程4 网络

part 1 网络？

I 各种网络命令

1）ipconfig

  • 语法：ifconfig <网络接口> <IP地址> [] [up] [down]
  • 用途：查看主机ip地址和其他一些网络接口信息

2) nslookup/host

  • 语法：nslookup <域名>
  • 用途：显示某个指定主机的ip地址，主机的域名作为该命令的命令行参数。

3） 其他命令

  1. hostname/uname:显示主机名

    • 语法：hostname uname[options]

  2. rwho:命令显示当前网络中所有登录用户的用户信息

  1. rusers:显示在局域网中登录的用户信息

    • rusers [options] [host_list]
    • options:
      • -a 显示所有的主机信息，即使当前没有任何用户使用该主机
      • -s 以长格式显示用户信息，显示格式与who命令相似

  1. ruptime:显示当前连入局域网中所有主机的状态

    • ruptiome[options]
    • options
      • -l 按照load average 排序
      • -t 按照machine uptime 排序
      • -u 按照number of users 排序

  1. ping：测试网络连接（向hostname发送一个IP数据包测试是否在网络上）

    • ping [options] hostname
    • options
      • -c count 发送并接受count个数据包
      • -f 每秒发送100个或相当于网络处理能力的数据包
      • -s packetsize 发送包的大小为packetsize，缺省就为56字节（加上8字节的包头）

  2. finger：显示user_list中的用户信息

    • finger [options] [user_list]

    • option

      • -m

      • -s
